# Image & Audio Preprocessing

This repository contains projects on **image and Audio preprocessing**, demonstrating data handling, cleaning, and preparation techniques using Python.


## Contents
- Image preprocessing pipeline using PyTorch
- Audio preprocessing pipeline using Librosa
- Visualizations of original and processed data
- Final tensors ready for ML/DL models

## Tools Used
- Python 3.12
- Google Colab
- PyTorch, Torchvision
- Librosa, Matplotlib
